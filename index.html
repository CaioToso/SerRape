Preguntas sobre la medicina del rapé

El rapé es una medicina tradicional indígena hecha a base de polvo de plantas y tabaco, utilizado para la limpieza energética y física.

Se utiliza para mejorar la concentración, liberar tensiones, abrir las vías respiratorias y conectar con la espiritualidad.

No se recomienda para personas con problemas cardíacos, presión alta o embarazadas. Siempre consultar con un especialista.
